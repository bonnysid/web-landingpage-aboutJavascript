Привет, дорогой друг!

JavaScript - это лёгкий, интерпретируемый, объектно-ориентированный язык с функциями первого класса, самый известный скриптовый язык для веб-страниц, но также используется во многих не браузерных окружениях. Для его использования необходимо установить NodeJS!

Площадь Треугольника

Здесь ты можешь посчитать площадь треугольника. В JavaScript вы можете сделать любую формулу, а результат округлить при помощи методов объекта Math.floor (в меньшую) и Math.ceil (всегда в большую).

Схожесть слов

Ты можешь проверить два слова на равенство их длин! У каждой строки есть свойство length, которое показывает количество символов в строке. Для строго сравнение используется '==='!
